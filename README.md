# Projeto Aplicado III: Sistema de Recomendação para Produtos Fitness

## Descrição
Desenvolvimento de um sistema de recomendação para produtos fitness utilizando o dataset de avaliações da Amazon. O objetivo é identificar padrões de consumo e sugerir produtos que possam ser relevantes para os usuários por meio da aplicação de técnicas de ciência de dados.

## Objetivos
* Coleta e preparação da base de dados pública (Amazon Product Reviews Dataset).
* Limpeza e tratamento dos dados.
* Análise exploratória focada em padrões de consumo de produtos fitness.
* Desenvolvimento e avaliação de um modelo de sistema de recomendação.
